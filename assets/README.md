# Assets

Drop two files here to fully personalize the site:

- `profile.jpg` — square portrait, ~600×600 px (used in the About section)
- `resume.pdf` — your resume (powers the "Download resume" button)

Both are optional. If `profile.jpg` is missing the site falls back to "SM" initials. If `resume.pdf` is missing the download link will 404, so just leave it out until you have it.
